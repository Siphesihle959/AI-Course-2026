# AI-Course-2026
Practical lab notebooks and mini-projects for the Artificial Intelligence module at Vaal University of Technology, offered as part of the Advanced Diploma in Computer Systems Engineering.

## What this repository covers
This repo is organized into 12 weeks of hands-on practicals, progressing from data handling to modern AI systems.

| Phase | Weeks | Focus |
|-------|-------|-------|
| I     |  1–3  | Python stack, NumPy, Pandas, data wrangling |
| II    |  4–6  | Classical ML: regression, classification, clustering |
| III   |  7–8  | Neural networks and CNNs with PyTorch and TensorFlow |
| IV    |  9–10 | NLP, transformers, and large language models |
| V     | 11–12 | AI ethics and capstone project |

## Tools and frameworks

- **Python** — core language
- **NumPy / Pandas** — data handling and wrangling
- **Scikit-Learn** — classical machine learning
- **PyTorch / TensorFlow** — deep learning
- **Jupyter Notebook** — all labs are in notebook format

## Capstone project

The final project applies the full AI workflow end-to-end. Options include an image classifier, an NLP sentiment analyzer, a recommender system, or a fine-tuned transformer model.
